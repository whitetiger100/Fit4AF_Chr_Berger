ACTIONS:
	Im Zusammenhang mit Forumlaren kennt Rails vier Actions:
		new: ein neuer Event wird im Controller erstellt und an eine Webtemplate views/event/new.html.erb weitergeleitet.
		create: die Daten eines neu erstellten Events werden zur CREATE-Action geleitet und dirt in der Datenbank gespeichert.
		edit: ein vorhandener Event wird in der Action edit und dann im Template views/events/edit.html.erb dargestellt.
		update: hier werden die Änderungen gespeichert.

HELPER:
	Formhelper zur Erstellung des Formulars
		<%= form_for(@event) do |f| %>
			...
		<% end %>
		Dieser Helper erstellt das Html-Formular. Die Methode 'form_for' erwartet also ersten Parameter ein Datenbank-Objekt.
		In der do-Schleife werden dann die einzelnen Tabellenspalten durchlaufen.
	Formhelper für die Formularelemente
		<%= f.text_field :name %>
		Dieser Helper 'text_field' erstellt ein Textfeld, dass auf die Tabellenspalte 'name' verweist.
